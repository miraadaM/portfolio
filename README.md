Mirada Makhmutova Portfolio Website
A personal portfolio website showcasing my projects, experience, and work. Built with HTML, CSS, and minimal JavaScript, featuring smooth animations and a light/dark mode toggle.
Features
About Me: Introduction and current work.
Projects: Showcases coding and research projects (some “In Progress”).
Experience: Work,leadership roles.
Contact: Get in touch via GitHub, LinkedIn, or Email.
Animations: Page content fades in smoothly when opening.
Light/Dark Mode: Toggle between sun/moon themes for the whole website.
Multi-page Layout: Easy navigation between sections.

How to Use / Run
Clone the repository:
git clone https://github.com/yourusername/portfolio.git
Open any page (index.html, projects.html, etc.) in a browser.
Click the Sun/Moon button to switch between light and dark modes.
Notes
Animations are handled with CSS keyframes, no external libraries required.
Theme preferences are stored using localStorage for persistence across reloads.
Fully responsive and optimized for desktop and mobile viewing.
